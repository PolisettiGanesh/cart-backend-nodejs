# Cart Backend (Node.js)

E-commerce backend application built using Node.js, Express, and MongoDB.

## Features (Day 1)
- Express server setup
- MongoDB connection using Mongoose
- User model with schema validation
- Create User API
- Duplicate email check

## Tech Stack
- Node.js
- Express.js
- MongoDB
- Mongoose
